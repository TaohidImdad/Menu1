5. Line Height and Letter Spacing

Properties: line-height, letter-spacing

line-height changes vertical spacing between lines. letter-spacing changes space between letters.

6. Google Fonts Usage

Use the Google Fonts website to choose fonts.

Example of using Google Fonts (Pacifico and Roboto):

<link href="https://fonts.googleapis.com/css2?family=Roboto&family=Pacifico&display=swap" rel="stylesheet">

Then apply with CSS:

font-family: 'Pacifico', cursive;

This is Pacifico font from Google Fonts.
