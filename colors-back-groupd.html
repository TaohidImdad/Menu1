CSS Colors & Backgrounds - Notes & Examples

1. Color Values

CSS supports different ways to define color:

  • Name: red, blue, green
  • Hex: #ff0000, #00ff00
  • RGB: rgb(255, 0, 0)
  • HSL: hsl(0, 100%, 50%)

2. Background Properties

  • background-color: sets solid background color
  • background-image: sets image background
  • background-repeat: repeat, no-repeat, repeat-x/y
  • background-position: top, center, right, etc.

3. Gradients

Gradients are smooth transitions between colors.

  • linear-gradient(): Top to bottom, left to right
  • radial-gradient(): Circle-like from center outward
