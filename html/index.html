HTML Syllabus

1. Introduction to HTML

  • What is HTML?
  • Structure of an HTML Document
  • <!DOCTYPE html>, <html>, <head>, <body>

2. Text Formatting Tags

  • Headings: <h1> to <h6>
  • Paragraph: <p>
  • Bold, Italic, Underline: <b>, <i>, <u>
  • Line Break and Horizontal Line: <br>, <hr>
  • Preformatted Text: <pre>

3. HTML Elements

  • Div and Span: <div>, <span>
  • Comments: <!-- comment -->

4. Lists

  • Ordered List: <ol>
  • Unordered List: <ul>
  • List Item: <li>
  • Definition List: <dl>, <dt>, <dd>

5. Links and Images

  • Anchor Tag: <a href="url">
  • Image Tag: <img src="image.jpg" alt="desc">

6. Tables

  • Table: <table>
  • Table Row: <tr>
  • Table Data: <td>
  • Table Header: <th>
  • Table Caption: <caption>

7. Forms and Input

  • <form>, <input>
  • Types: text, password, email, number, radio, checkbox, submit
  • <label>, <textarea>, <select>, <option>
  • <fieldset>, <legend>

8. Semantic Elements

  • <header>, <footer>
  • <section>, <article>, <aside>
  • <nav>, <main>

9. Multimedia

  • Audio: <audio controls></audio>
  • Video: <video controls></video>
  • Embed: <embed>, <iframe>, <object>

10. HTML5 APIs (Intro only)

  • Canvas: <canvas>
  • Geolocation API
  • Local Storage: localStorage

11. Best Practices

  • Indentation and readability
  • Using semantic tags
  • Accessibility (alt tags, labels)
  • Validation using W3C validator
