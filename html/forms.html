HTML Forms - Full Notes & Examples

<form>

The container for all form elements.

Attributes:

  • action: URL where form data is sent
  • method: get or post
<form action="/submit" method="post">...</form>

<fieldset> and <legend>

Groups related form elements.

Personal Info

<label>

Defines a label for a form element. Use for attribute to link to input ID.

<input> Element and Types

Used to take user input in various formats.

  • type="text"
  • type="email"
  • type="password"
  • type="checkbox"
  • type="radio"
  • type="number"
  • type="submit"
  • type="reset"

<select>, <option>, <optgroup>

Dropdown list with groupings:

<textarea>

Multi-line text input.

<button>

Clickable button for submit or any JavaScript action.

Common Input Attributes

  • type – Type of input (text, checkbox, radio, etc.)
  • name – Name for form submission
  • value – Default value
  • placeholder – Placeholder text
  • required – Makes field mandatory
  • readonly – Prevents editing
  • disabled – Disables the field
  • maxlength, min, max, etc.
