HTML Tabular Data - Full Notes

1. <table>

Creates a table structure to display tabular data.

<table>...</table>

2. <caption>

Provides a title for the table.

<caption>Student Marks</caption>

3. <colgroup> and <col>

Used for styling specific columns.

<colgroup>
  <col span="1" style="background-color:lightgray;">
  <col span="2" style="background-color:lightblue;">
</colgroup>

4. <thead>, <tbody>, <tfoot>

These tags help organize the structure of a table:

  • <thead> - Header rows
  • <tbody> - Main content rows
  • <tfoot> - Footer rows

5. <tr>, <th>, <td>

  • <tr> - Table row
  • <th> - Table header cell (bold, centered)
  • <td> - Table data cell (normal cell)

6. Full Table Example

Here is a full example using all tabular tags:

Employee Report
ID Name Department
101 John HR
102 Jane IT
End of Report

7. Tips

  • Use scope="col" and scope="row" in <th> for accessibility
  • Style your tables with CSS or inline styles
  • Use <colgroup> for column-specific styles
